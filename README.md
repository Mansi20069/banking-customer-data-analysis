Banking CustomerData Analysis 
Project overview: This project focus on analysing banking data to extract meaningful insight little two customer behaviour financial.
This project focuses on analyzing banking data to understand customer behavior, financial trends, and risk levels using Power BI. The dataset includes customer details, account information, transaction records, and risk categories.
The data is prepared and organized within Power BI, followed by the creation of interactive dashboards using charts and visual tools. These dashboards help in identifying high-value customers, analyzing transaction patterns, and understanding overall financial performance.
The project provides useful insights that support better decision-making and effective risk management in the banking sector. It also helps in improving customer segmentation and tracking key performance indicators through clear and interactive visual reports.
